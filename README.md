# Desafio Azure AI: Reconhecimento de Fala

## Introdução

Este projeto faz parte de um desafio para explorar e aplicar os recursos de Inteligência Artificial da Microsoft Azure, especificamente o serviço de Fala (Speech).

## Ferramentas Utilizadas

* **Microsoft Azure Speech Studio:** Plataforma da Azure para desenvolver e integrar funcionalidades de fala.

## Etapas Realizadas

1.  **Criação do Recurso de Fala:** Configurei um novo recurso de "Serviço de Fala" no Portal do Azure.
2.  **Upload e Transcrição de Áudio:** Utilizei o **Speech Studio** para fazer o upload de um arquivo de áudio e, em seguida, o sistema o transcreveu automaticamente para texto.
